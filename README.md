<h1 align="center">ASP.NET MVC Form Handling Demo</h1>

A simple and elegant ASP.NET MVC web application demonstrating form handling, model validation, TempData, and Session state management.  
Users can submit their **Name**, **Age**, and **Favorite Programming Language**, and the app displays the results using a clean UI with animated glassmorphism styling.

✨ **Features:**  

1. MVC-based form handling (Model → View → Controller)  
2. Server-side validation using Data Annotations  
3. TempData for one-time value storage (Name, Age)  
4. Session for persistent user data (Favorite Language)  
5. Modern animated UI with gradient background and glass effect  
6. Fully responsive and beginner-friendly project structure  
7. Perfect for students learning ASP.NET MVC, state management, and clean UI integration
