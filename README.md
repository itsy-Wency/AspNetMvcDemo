A simple and elegant ASP.NET MVC web application demonstrating form handling, model validation, TempData, and Session state management.
Users can submit their Name, Age, and Favorite Programming Language, and the app displays the results using clean UI with animated glassmorphism styling.

✨ Features:

MVC-based form handling (Model → View → Controller)

Server-side validation using Data Annotations

TempData for one-time value storage (Name, Age)

Session for persistent user data (Favorite Language)

Modern animated UI with gradient background and glass effect

Fully responsive and beginner-friendly project structure

Perfect for students learning ASP.NET MVC, state management, and clean UI integration.
